기능
파티에 들어가거나 파티원이 들어오면 1~4층 프프로그 띄워주고
데이터베이스에 박제돼있는지 보여줌


사용법

1. 오른쪽 상단의 Code를 누르고 Download Zip을 눌러 다운받고 압축 풀기
2. ACT -> Plugins -> FFXIV Settings -> Enable Debug Option -> Log all Network Packets
3. ACT -> Plugins -> OverlayPlugin.dll
4. 왼쪽 하단의 추가 클릭 -> 이름은 아무렇게, Preset은 커스텀, 유형은 Mini Parse
5. URL 오른쪽의 ...을 누르고 받은 html파일을 넣기
6. 하다 오류나면 파탈하고 새로고침 누르셈 ㅋㅋ
------------------

하다 오류나면 상세한 상황 제보해주셈.
